How To Apply for Class IV (Peon)/ Group D Vacancy 2025 :
  • आधिकारिक वेबसाइट पर जाएं: सबसे पहले hcraj.nic.in (राजस्थान हाई कोर्ट की आधिकारिक वेबसाइट) पर जाएं। वहां मेन्यू में दिए गए Recruitment सेक्शन पर क्लिक करें।
  • Apply Online लिंक पर क्लिक करें: Class IV (Peon) Recruitment 2025 से संबंधित विज्ञापन पर क्लिक करें और फिर Apply Online लिंक खोलें।
  • पंजीकरण करें (Registration): नई उम्मीदवार हों तो New Registration पर क्लिक करें। अपना नाम, मोबाइल नंबर, ईमेल आईडी और अन्य जरूरी जानकारी भरें। सिस्टम आपको एक User ID और Password देगा – इसे सुरक्षित रखें।
  • लॉगिन करें और आवेदन फॉर्म भरें: User ID और Password के माध्यम से लॉगिन करें। अब आवेदन फॉर्म खुलेगा – इसमें अपनी व्यक्तिगत जानकारी, शैक्षणिक योग्यता, पता आदि भरें।
  • दस्तावेज़ अपलोड करें: पासपोर्ट साइज फोटो, हस्ताक्षर (Signature), जाति प्रमाण पत्र (यदि लागू हो), निवास प्रमाण पत्र आदि इन सभी को निर्धारित फॉर्मेट और साइज में अपलोड करें।
  • आवेदन शुल्क जमा करें: शुल्क का भुगतान ऑनलाइन माध्यम से करें
  • फॉर्म सबमिट करें: सभी जानकारी ध्यानपूर्वक जांचने के बाद Submit बटन पर क्लिक करें।
  • प्रिंट निकालें: आवेदन सफलतापूर्वक जमा होने के बाद उसका प्रिंटआउट निकाल लें और भविष्य के लिए सुरक्षित रखें।
